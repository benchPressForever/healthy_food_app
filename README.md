# Мобильное приложение для подсчёта КБЖУУ

Данный репозиторий представляет собой проект мобильного приложения для подсчёта калорий и нутриентов (КБЖУ), разрабатываемый в рамках курса **«Технологии разработки программных средств»**.

---

## О проекте

**Цель проекта** — создание полнофункционального мобильного приложения, которое помогает пользователям отслеживать суточное потребление белков, жиров, углеводов и калорий, вести дневник питания и достигать целей по изменению веса.
